### ANOMALIE : la racine "d" est FONCTIONNELLE

- d+aiin = 587x (48% de tous les d-mots) vs ch+aiin = 40x (4%)
- d ne prend PAS -edy ni -eedy (0x !)
- d est probablement une PARTICULE (preposition? article?)
- Les vraies racines lexicales sont ch, sh, cth, ckh, kch, dch, otch

### 2 CLASSES DE RACINES

**FONCTIONNELLES** (entropie < 2.0, concentrees sur 1-2 suffixes) :
- cho, sho, cheo, sheo, qokeo : quasi-exclusivement -dy (87-98%)
- d, chod, shod : massivement -aiin

**LEXICALES** (entropie > 2.5, distribution equilibree) :
- ch, sh, ok, qok, ot, k, yk, yt, cth, ckh, kch, dch, tch, otch
- Prennent 10-14 suffixes chacune

### ANTI-CORRELATION -ol / -edy

Les pages herbal se divisent en 2 groupes :
- Groupe "ol" (31 pages, AVEC chol.daiin) : -edy quasi-absent (ratio 0.17)
- Groupe "edy" (117 pages, SANS chol.daiin) : dominant -edy

Les memes racines prennent -ol OU -edy selon le groupe de pages.
C'est un clivage de REGISTRE ou de TYPE DE CONTENU.

### CHAINES de suffixes (trigrams)

- edy > edy > edy (124x) = listes/enumerations
- ol > ol > ol (92x) = descriptions
- ol > aiin > ol (44x) = "qualite ET qualite" ?
- MAIS daiin relie 80% du temps des suffixes DIFFERENTS
  => daiin n'est probablement PAS "et"

### -am = TERMINATEUR DE PHRASE

71% des mots en -am sont le DERNIER mot de la ligne.
Les segments entre deux -am ont une longueur mediane de 30 mots.
C'est la PONCTUATION du VMS.

## CE QUE CHOL.DAIIN SIGNIFIE PROBABLEMENT

Decomposition : ch[racine lexicale] + ol[suffixe descriptif] + d[particule] + aiin[connecteur]

34 occurrences, TOUTES dans le herbal, toutes sur 31 folios specifiques.
Position 0.50 (milieu de ligne), contexte symetrique.

Les pages AVEC chol.daiin sont les pages du registre "-ol" (descriptif).
Les pages SANS sont le registre "-edy" (procedural).

Hypothese la plus probable : chol.daiin = formule de description galenique
("calidum in" ou "de nature chaude" ou similaire)

## HYPOTHESES CLASSEES

### H1 (50%) — Suffixes de REGISTRE textuel
- -ol/-or = DESCRIPTIF (proprietes, nature de la plante)
- -edy/-eedy = PROCEDURAL (instructions, listes d'ingredients)
- -aiin/-ain = CONNECTIF (liaisons)
- -am = CONCLUSIF (fin de phrase)

### H2 (30%) — Marqueurs de CAS (declinaison simplifiee)
- -ol = ablatif, -or = nominatif, -edy = genitif, -ar = accusatif, -am = acc.fem.
- Probleme : d prend TOUS les cas, pas un seul comme en latin

### H3 (15%) — Marqueurs THEMATIQUES
- -ol = identite, -or = origine, -edy = procedure, -ar = finalite, -al = quantite

### H4 (5%) — Variantes sans semantique fixe
- Refute par la distribution positionnelle et sectionnelle
