# Prompt 9 — Noms de plantes vernaculaires italiens du XVe siecle

## Statut : A ENVOYER (PRIORITE HAUTE)

CONTEXTE : Nous dechiffrons le Manuscrit de Voynich (1404-1438) comme un receptaire pharmaceutique en latin chiffre. Le scribe utilise des noms ITALIENS vernaculaires (pepe = poivre, lilie = lis). Les premiers mots des pages herbales NE SONT PAS les noms de plantes en K&A phonetique (test negatif sur 7 plantes). Les noms d'ingredients dans les recettes utilisent les valeurs K&A minoritaires.

BESOIN PRECIS : Dictionnaire des noms de plantes medicinales en italien/toscan/lombard/veneto du XIVe-XVe siecle.

PRIORITE : Noms encodables par K&A (pas de b, f isole, g, z, doubles consonnes rr/ss/ll/mm).

3 SOURCES PRIORITAIRES PRE-LINNEENNES :
1. Simon Online (simonofgenoa.org) : XIIIe, synonymes en vulgaire italien, structure wiki parsable
2. Matthioli, Discorsi (1568) : commentaire Dioscoride en italien, Archive.org
3. Erbario LJS 419 : veneto XVe siecle — exactement le dialecte pertinent pour le VMS

FORMAT : Tableau nom latin medieval | nom italien toscan | nom venitien/lombard | encodable K&A (oui/non)
