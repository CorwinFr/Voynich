# Plan de Recherche — Validation Externe du Modèle V22-V34

## Objectif
Trouver des preuves EXTERNES pour valider (ou invalider) notre modèle de décodage.
Chaque prompt est autonome, ne référence aucun fichier local.

## 4 Axes de Recherche

### Prompt 1 : Recettes de l'Antidotarium Nicolai
Trouver les VRAIS textes de recettes médiévales pour comparer avec nos décodages.
→ Fichier : `prompt_01_antidotarium.md`

### Prompt 2 : Traitements galéniques par signe zodiacal
Vérifier si chaque signe zodiacal a des traitements spécifiques dans la médecine médiévale.
→ Fichier : `prompt_02_zodiac_galenic.md`

### Prompt 3 : Calendriers lunaires de prescription médiévaux
Vérifier si des volvelles/calendriers lunaires pharmaceutiques existaient au XVe siècle.
→ Fichier : `prompt_03_lunar_calendars.md`

### Prompt 4 : Système de notation pharmaceutique abrégé
Trouver des exemples de notation abrégée dans les manuscrits pharma médiévaux italiens.
→ Fichier : `prompt_04_notation_system.md`
