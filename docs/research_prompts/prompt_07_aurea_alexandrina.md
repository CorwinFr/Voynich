# Prompt 7 — Aurea Alexandrina : texte diplomatique complet

## Statut : A ENVOYER

CONTEXTE : Je travaille sur le dechiffrement algorithmique du Manuscrit de Voynich (MS 408, Beinecke Library, Yale). Notre pipeline de decodage (K&A v12) identifie le texte comme du latin pharmaceutique medieval. Le folio f103r (section recettes) presente 498 mots dont 91% valides en latin Perseus, avec les ingredients aloes, ture (encens), asari, olei, sal, aceto, mel, cera identifies. Nous pensons que ce folio encode une ou plusieurs recettes de l'Antidotarium Nicolai, possiblement l'Aurea Alexandrina (premiere recette du recueil).

BESOIN PRECIS : Je cherche le texte COMPLET, mot a mot, de la recette "Aurea Alexandrina" telle qu'elle apparait dans l'Antidotarium Nicolai. Pas un resume moderne, pas juste la liste d'ingredients — le texte DIPLOMATIQUE latin tel qu'un scribe du XVe siecle l'aurait lu.

CE QUE JE CHERCHE SPECIFIQUEMENT :
1. Le texte latin complet de l'Aurea Alexandrina depuis "Recipe..." jusqu'a la fin de la preparation
2. Si possible, les VARIANTES entre manuscrits
3. Le texte complet de 2-3 autres recettes courtes (Oximel, Hiera Picra, Metridatum)

SOURCES : Edition Goltz (1976), van den Berg (1917/dbnl.org), BNF Latin 6823 (Marco Ponzi/Medium), Brussels KB 15624-15641.
