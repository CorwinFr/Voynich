## CHAMPS SEMANTIQUES RECURRENTS DANS LES 4 LUNAIRES

### Theme MALADIE/GUERISON (tous les jours)
- guerir / gairira / sanabitur / sanare
- languir / languira / languebit
- mourir / mourra / morietur
- medecine / medicamen / potio
- peril / perilleus / periculum
- lever / se levera / surgere
- conforter / conferme / confortare
- sans peril / sine periculo
- peur de mourir / timor mortis
- longue enfermete / longa infirmitas

### Theme SAIGNEE (jours specifiques)
- saigner / sanguinem minuere
- avant tierce / ante tertiam
- mauvais sang / malus sanguis
- interdit / non bonum

### Theme ACTIVITES (jours pairs souvent favorables)
- acheter/vendre / emere/vendere
- semer/planter / seminare/plantare
- mariage / nuptiae / matrimonium
- voyage / iter / navigare
- construire / aedificare
- commerce / mercari
- entrer maison / domum intrare
- habits neufs / nova vestimenta

### Theme REVES
- veridiques / verus
- vains / vanus
- se realisent en X jours
- joyeux / gaudium

### Theme NAISSANCE
- caractere de l'enfant
- duree de vie
- marques de naissance
- destin (riche/pauvre, sage/fou)

### JOURS SYSTEMATIQUEMENT MAUVAIS : 3, 5, 13, 25
### JOURS SYSTEMATIQUEMENT BONS : 1, 4, 14, 17, 18, 29
