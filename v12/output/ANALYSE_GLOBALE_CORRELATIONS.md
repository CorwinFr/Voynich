## Correlations cles

1. **BIO/BAINS = meilleur decode** (87% connu, 3% inconnu) — vocabulaire le plus repetitif
2. **HERBAL = plus de doutes** (14%) — les noms de plantes sont un NOMENCLATEUR separe
3. **ZODIAC = plus d'inconnus** (15% doute + 12% inconnu) — noms de signes = nomenclateur
4. **RECETTES = bon decode** (81%) mais 11% doute = noms d'ingredients non resolus

## Le gap principal : le NOMENCLATEUR des plantes

Les 14% de doute dans l'HERBAL sont des NOMS DE PLANTES que K&A
ne decode pas. Ce sont probablement des noms en langue vernaculaire
(italien, provencal, arabe translittere) encodes avec un systeme
DIFFERENT de K&A.

Solution possible : croiser les illustrations de plantes avec le
nomenclator italien (4466 noms) en utilisant les identifications
botaniques existantes (INELIODE = Inula helenium, etc.).
