# DECOUVERTE : Le chiffre des ingredients

## Le systeme DUAL confirme

L'apothicaire utilisait les MEMES GLYPHES avec DEUX lectures :

### Mode STRUCTURE (mots de fonction)
Chaque glyphe = un MOT LATIN complet
```
o = ac (et)
l = se (soi)  
d = de (de)
r = recipe (prends)
...
```

### Mode INGREDIENT (noms de substances)  
Chaque glyphe = une LETTRE LATINE
```
o = a
l = s
t = l
e = o
r = r
a = a
i = r/u
n = m
```

## Preuves (4 ancres)

| Latin | EVA | Mapping confirme |
|-------|-----|-----------------|
| **a**loe | **o**teey | o=a, t=l, e=o, y=e |
| **a**s**a**r | **o**l**o**r | o=a, l=s, r=r |
| **a**s**a**rum | **o**l**a**iin | a=a, i=r, n=m |
| **a**s**a**ra | **o**l**a**r | confirmation |

## Mapping partiel (8/17 valeurs)

| EVA glyph | Structure mode | Ingredient mode |
|-----------|---------------|-----------------|
| o | ac (et) | **a** |
| l | se (soi) | **s** |
| t | el (le) | **l** |
| e | et (et) | **o** |
| r | recipe | **r** (identite) |
| a | ana | **a** (identite) |
| i | 1 (chiffre) | **r** ou **u** |
| n | drachme | **m** |

## Glyphes NON resolus
