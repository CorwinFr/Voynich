## 1. Textes latins de recettes : 10 exemples verifies

Les textes latins ci-dessous proviennent de l'editio princeps (Nicolas Jenson, Venise 1471) accessible via DBNL (van den Berg 1917) et des transcriptions de Marco Ponzi (BNF Latin 6823, Italie du Sud, XIVe siecle).

### 1.1 Electuarium de succo rosarum
- **Texte** : "Recipe ... ana ... tempera ad modum electuarii ... datur cum aqua calida in mane"
- **Ingredients** : zuchari ana lb.i et ℥ iiii ; succi rosarum ana lb.i et ℥ iiii ; sandali albi, sandali rubei, sandali citrini ana ʒ vi ; spodii ʒ iii ; diagridii ʒ xii ; camphore scr. i
- **Verbes** : Recipe -> tempera -> (syrupe facto) -> datur
- **Connecteurs** : ana (equivalence), et, cum (vehicule : cum aqua calida), in (temps : in mane)
- **Source** : DBNL editio princeps

### 1.2 Oleum rosatum
- **Texte** : "... oleum ... sic fit ... ponatur ... in olla ... in caldario pleno aqua ... bulliant ... exprimatur ... reservetur"
- **Ingredients** : olei communis (ablutus) duae librae ; rosarum viridium aliquantulum contusar(um) lb.i et media
- **Verbes** : sic fit -> ponatur -> bulliant -> exprimatur -> reservetur
- **Connecteurs** : in (recipient : in olla, in caldario), quousque (jusqu'a)
- **Source** : DBNL editio princeps

### 1.3 Oximel
- **Texte** : "OXimel sic fit. Recipe ... in quo bulliant ... deinde coletur: et colatura ... addatur ... bulliat"
- **Ingredients** : mellis despumati lb.i ; aceti fortissimi lb.i et ½ ; radices feniculi ʒ ii ; radices rafani ʒ ½ ; radices apii/petroselini/macedonici/sparagi/bruxi/saxifrag(iae) an ʒ iiii ; polipodii ʒ viii
- **Verbes** : Recipe -> bulliant -> coletur -> addatur -> bulliat -> datur
- **Connecteurs** : in quo (milieu de decoction), deinde, et, cum (cum calida = vehicule)
- **Source** : DBNL editio princeps

### 1.4 Oxizaccare
- **Texte** : "Recipe zuchari ... succi malorum granatorum ... aceti ... pone ... bulliat ... donec ... detur in mane cum calida"
- **Ingredients** : zuchari lb.i ; succi malorum granatorum ʒ viii ; aceti ʒ iiii
- **Verbes** : Recipe -> pone -> bulliat -> donec -> detur
- **Source** : DBNL editio princeps

### 1.5 Yera pigra Galyeni (Hiera Picra)
- **Texte** : "YErapigra ... Recipe ... an.℥.ii ... aloes ... mel quod sufficit. dosis eius .ʒ.iii"
- **Ingredients** : cinnamomi, spice, croci, squinanti, asari, xilocassie, xilobalsami, carpobalsami, viole, absinthii, agarici, rosarum, turbit, coloquintide, masticis an ℥ ii ; aloes "ad pondus omnium specierum" ; mel quod sufficit
- **Verbes** : Recipe -> (implicite conficere par le miel) -> datur
- **Dose canonique** : ʒ iii (3 drachmes), PAS 3 onces
- **Source** : DBNL editio princeps ; Ponzi (viridisgreen/Medium)

### 1.6 Mel rosaceum (Rodomel)
- **Texte** : "sic fit ... ponatur ... ad ignem ... cum bullire inceperit ... addantur ... bulliant ... agitetur"
- **Ingredients** : mellis albissimi et purissimi (dispumati) librae x ; succi recentium rosarum viridium incisarum librae iiii
- **Verbes** : sic fit -> ponatur -> inceperit bullire -> addantur -> bulliant -> agitetur
- **Source** : DBNL editio princeps

### 1.7 Unguentum fuscum
- **Texte** : "Recipe olei ... cere ... Confice sic: oleum ... bulliat: tunc apponatur cera ... deinde ... donec"
- **Ingredients** : olei lb.i et ½ ; cerae ʒ iiii ; colofoniae ʒ ii ; gummi serapini an ʒ iii ; picis navalis colatae an ʒ iii ; mastici an ʒ i ; galbani an ʒ i ; terebintinae an ʒ i
- **Verbes** : Recipe -> Confice sic -> mittatur -> bulliat -> apponatur -> liquefiant -> agitando -> reservetur
- **Connecteur cle** : tunc (sequencage)
- **TRES PERTINENT** : base oleum + cera, exactement le motif decode
- **Source** : DBNL editio princeps

### 1.8 Unguentum album
- **Texte** : "pulvis ... distemperetur ... addatur ... aqua rosarum ponatur ... cum inceperit inspissari addatur oleum et aqua rosarum"
- **Ingredients** : cerusae ʒ ii ; litargiri ʒ i ; olibani ʒ iii ; masticis ʒ ii
- **Verbes** : Recipe -> teratur -> distemperetur -> addatur -> ponatur -> inceperit -> ducatur -> recondatur
- **Motif** : additions alternees de aqua rosarum et d'oleum = vehicule + base
- **Source** : DBNL editio princeps

### 1.9 Metridatum (Mithridate)
- **Texte** : "MEtridatum ... Recipe ... an.ʒ.i ... an.℥.ii ... mel quod sufficit. datur in sero in modum avellane"
- **Ingredients** : 60+ ingredients organises en blocs de quantite : Bloc A (quantite propre) storacis calamite ʒ i + ℥ i ; Bloc B (an = ʒ i) ~40 substances ; Bloc C (an = ℥ ii) ~6 substances ; Bloc D (an = ℥ i) ~30 substances
- **Structure** : le mot "an" (ana) segmente fortement la liste en blocs de dosage identique
- **Administration** : in modum avellane (taille d'une noisette), avec vin cuit et sauge
- **Source** : DBNL editio princeps

### 1.10 Tyriaca magna Galeni (Grande Theriaque)
- **Texte** : "TYriaca magna Galeni ... Recipe ... an ... diversa est eius dosis"
- **Ingredients** : trociscorum squilliticorum ʒ ii + ℥ ii ; piperis longi ʒ ii ; trociscorum tyri et diacoralli an ʒ i + ℥ i ; xilobalsami ℥ i (+ gr vii dans un temoin) ; opii, agarici, yris, rose, scordei, seminis rape silvestris, cinamomi, opobalsami an ℥ i + gr xiv
- **Verbes** : Recipe -> (liste) -> datur
- **Source** : DBNL editio princeps

## 2. Motifs structurels du latin pharmaceutique medieval

### 2.1 Ordre des mots
Le verbe initial **Recipe** ouvre la section prescriptive, suivi d'une serie nominale au genitif (mellis despumati, aceti fortissimi, olei communis), puis des quantites.

**Schema canonique** : Recipe + [INGREDIENT genitif] + [QUANTITE] + [VERBE procedural] + ... + datur/detur [MODE D'ADMINISTRATION]

### 2.2 Position des quantites
Les quantites sont placees majoritairement APRES l'ingredient : "mellis ... libram i", "olei communis ... duae librae". Deux schemas coexistent : (a) "ingredient - quantite" en serie, (b) "liste d'ingredients + an + quantite" (ana groupant les ingredients a dose egale).

### 2.3 Connecteur "et"
Sert a chainer ingredients ET operations ("deinde ... et ... cum"). Mais une grande part de l'enumeration est **asyndetique** (simples juxtapositions sans connecteur) avant "an".

### 2.4 Connecteur "cum" : double emploi
- **cum = vehicule d'administration** : datur cum aqua calida, cum vino calido, cum decoctione - extremement frequent
- **cum = mode de preparation/melange** : mixtum cum masticis..., temperetur cum aceto et oleo - introduit l'excipient/solvant

### 2.5 Morpheme "in"
Introduit principalement :
- Un **recipient/dispositif technique** : in olla, in caldario pleno aqua suspensa, in stagnato vase
- Un **repere temporel d'administration** : in mane, in sero

### 2.6 Marqueurs de procede
L'Antidotarium emploie des **marqueurs de procede** plutot que des tags abstraits M/P : despumatus (ecume), ablutus (lave), contusus (broye), incisus (coupe), teratur (broyer), cribrari (tamiser), coquere/bullire (cuire/bouillir), coletur/cola (filtrer).

## 3. Validation des motifs decodes contre les pratiques attestees

### 3.1 "cum AQUA" comme vehicule
**VALIDE.** Parfaitement standard dans l'Antidotarium. Apparait dans des cadres de posologie (datur cum aqua calida in mane) ou comme eau specialisee (aqua rosarum, aqua pluvialis). L'Electuarium de succo rosarum se donne explicitement cum aqua calida.

### 3.2 Sequence "Rx ... misce, cola"
**VALIDE.** L'enchainement Recipe -> (bullire/coque) -> coletur/cola -> addatur -> bulliat est un modele tres commun. L'Oximel illustre parfaitement : bulliant -> coletur -> addatur -> bulliat.

### 3.3 Distinction [M]MATERIA vs [P]MATERIA
**PARTIELLEMENT VALIDE.** La distinction brut/prepare est bien exprimee dans l'Antidotarium, mais par des **adjectifs/participes** (recens, viridis, ablutus, contusus, despumatus, incisus), pas par un systeme de prefixe categoriel M/P. Exemples : "oleum ... ablutus" et roses "contusae" dans l'Oleum rosatum ; "cera preparata" ou "cera lota" dans les onguents. La conceptualisation est identique, le codage est different.

### 3.4 "2℥ dose dominante"
**NON CONFIRME comme norme universelle.** L'Antidotarium donne souvent des doses en drachmes (ʒ) ou en taille alimentaire (in modum avellane / castanee). La dose de 2 onces n'est pas "la" norme, mais 2℥ apparait comme quantite d'ingredient individuel dans de nombreuses recettes.

### 3.5 "HIERA 3℥"
**ATTENTION : confusion d'unites probable.** La dose de Hiera picra dans le temoin DBNL est "dosis eius ʒ iii" = 3 DRACHMES, pas 3 onces. 3 onces (= 24 drachmes !) de Hiera Picra seraient une quantite enorme et potentiellement dangereuse. Le decodage devrait etre reinterprete.

### 3.6 ACETUM pour applications externes
**VALIDE.** Bien atteste : un onguent "contre serpigo/elephantia" est tempere cum aceto et oleo avant onction. Le vinaigre est un vehicule/solvant topique reconnu, utilise en gargarismes, compresses, cataplasmes.

### 3.7 OLEUM + CERA ensemble
**FORTEMENT VALIDE.** Le binome oleum + cera est la base de l'Unguentum fuscum (= structure tres stable d'onguent). La sequence commence par olei et cerae, puis deroule une sequence chauffee (bulliat ... apponatur cera). C'est la definition meme du **cerat** (ceratum = cire + huile).

### 3.8 ANA en fin de liste
**PARTIELLEMENT VALIDE.** Ana est plutot utilise "au milieu" : on enumere une serie d'ingredients, on marque "an" puis la quantite, ce qui segmente la liste en blocs de dosage identique. Le Metridatum illustre cela avec ses blocs an ʒ i / an ℥ ii / an ℥ i.

### 3.9 Motif compose oleum + cera + sal + aqua
**NON CONFIRME comme motif unique.** Les quatre ingredients ne co-apparaissent pas dans une meme recette attestee. On trouve frequemment : (i) oleum+cera (onguents), (ii) cum aqua (posologie), (iii) sal (dans diverses confectio/electuaires). Le cas le plus proche est l'Oleum rosatum (oleum, cera dans un contexte, aqua rosarum comme bain-marie), mais sans sel.

## 4. Structure du Circa Instans (Platearius)

### 4.1 Organisation
- **Lemme** : nom du simple (organisation alphabetique)
- **Complexion galenique** : "X est calidum/frigidum et siccum/humidum in N gradu" (degre 1-4)
- **Vertus** : "virtutem habet..." (effets/indications)
- **Usages** : parfois avec remarques de preparation et d'administration

### 4.2 Exemple type
"Aloe calide et sicce complexionis est in secundo gradu. Aloe ex succo..." (British Library ; Cambridge MS Gonville and Caius 345/620)

### 4.3 Degres galeniques
Format typique : calidum/calid(a) et siccum/sicca + in primo/secundo/tertio/quarto gradu. Parfois "temperate in primo gradu" pour un effet modere.

### 4.4 Preparations
Beaucoup d'articles sont surtout descriptifs (qualite/vertu), mais certains ajoutent des modalites d'usage (decoction, quantites, administration). Exemple dans un corpus du Circa Instans : "Eupatorium calidum est in primo gradu, siccum in ii" suivi d'usages et d'une mini-preparation/dose (coction et administration en ʒ/ʒ).
